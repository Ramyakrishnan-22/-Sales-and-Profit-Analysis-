 #  Methodology

Cleaned the dataset (removed blanks and inconsistencies)

Converted the dataset into an Excel Table

Created pivot tables for sales and profit analysis

Added a calculated column:

Profit Margin = Profit / Sales


Designed interactive charts

Added slicers for Region, Category, and Segment

Applied conditional formatting for performance highlighting

Exported the final dashboard as a PDF report
